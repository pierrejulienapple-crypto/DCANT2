Politique de confidentialité

DCANT — Application de calcul de marge
Dernière mise à jour : mars 2026

1. Responsable du traitement

Le responsable du traitement est l'éditeur de DCANT. Contact : pierrejulienapple@gmail.com

2. Données collectées et pourquoi

Données de compte (nécessaires au service) : adresse e-mail et identifiant, utilisés uniquement pour vous authentifier et associer vos calculs à votre compte. Ces données ne sont jamais utilisées dans le benchmark.

Données de calcul : domaine, cuvée, millésime, prix d'achat, charges, marges et prix de vente que vous saisissez ou importez.

Documents importés : factures fournisseurs (PDF ou images) transmis à l'API Anthropic pour extraction automatique. Non conservés après traitement.

Données analytiques (avec votre consentement) : comportement de navigation via Microsoft Clarity (heatmaps, sessions anonymisées). Activé uniquement si vous acceptez les cookies.

3. Le benchmark anonymisé — comment ça fonctionne

Si vous y consentez explicitement, certaines données de vos imports participent à un benchmark collectif. Voici exactement ce qui se passe :

  • Seuls trois champs sont utilisés : domaine, millésime, prix d'achat.
  • Votre identité (e-mail, identifiant) n'est jamais incluse dans ces données agrégées.
  • Les données ne sont affichées qu'agrégées (médiane sur minimum 5 contributeurs). Impossible de retrouver votre prix individuel.
  • Ce consentement est facultatif, séparé du service, et révocable à tout moment depuis la page Benchmark.

En clair : DCANT sait qui vous êtes pour vous connecter, mais les autres utilisateurs ne voient que des médianes anonymes.

4. Sous-traitants

  • Supabase (USA) — base de données et authentification. Politique.
  • Vercel (USA) — hébergement. Politique.
  • Anthropic API (USA) — analyse IA des documents importés. Non conservés après traitement. Politique.
  • OpenAI Whisper API (USA) — transcription audio. Non conservés après traitement. Politique.
  • Microsoft Clarity (USA) — analytics comportemental, uniquement avec consentement. Politique.

Transferts vers les USA encadrés par les clauses contractuelles types de la Commission européenne.

5. Base légale

  • Exécution du contrat (art. 6.1.b RGPD) : e-mail, calculs.
  • Consentement (art. 6.1.a RGPD) : cookies Clarity, benchmark.
  • Intérêt légitime (art. 6.1.f RGPD) : sécurité.

6. Durée de conservation

  • Compte et calculs : durée d'utilisation + 12 mois après fermeture.
  • Documents importés : non conservés après traitement IA.
  • Données Clarity : 13 mois maximum.

7. Vos droits

Accès, rectification, effacement, limitation, portabilité, opposition, retrait du consentement. Contact : pierrejulienapple@gmail.com. Réclamation possible auprès de la CNIL.

8. Cookies

Cookies strictement nécessaires (session, préférences) : pas de consentement requis. Cookies analytiques (Microsoft Clarity) : consentement demandé à la première visite, modifiable à tout moment.
